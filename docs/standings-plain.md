# Standings (plain-English)

This page explains how the standings table works without technical terms.

## The big idea

- Each player earns results from every game.
- The standings show who is doing well and who has played more games.
- Everything is based on the scores that are entered after each match.

## What gets tracked

- Wins and losses.
- Total points scored.
- Total points allowed.
- Point differential (points scored minus points allowed).

## How the standings feel fair

- Players are compared using the same stats.
- Point differential helps separate tied records.
- Everyone’s totals only grow when a real score is entered.
